# BookWorm application
Application made with Vue.js and Bootstrap for tracking your favourite books.

## Features

- **User Authentication**: Login and register
- **Book Browsing**: Search and filter books by author and genre
- **Personal Library**: Add books to your library with reading status (Want to Read, Currently Reading, Read)
- **Admin Panel**: Admin users can add and remove books from the list
- **Responsive UI**: Built with Bootstrap 5 and Vue 3

## Tech Stack

- **Frontend**: Vue 3 with Composition API
- **Routing**: Vue Router
- **Styling**: Bootstrap 5 with custom SCSS
- **Build Tool**: Vite
- **Authentication**: JWT tokens stored in localStorage

## Getting Started

### Prerequisites

- Node.js 20 or higher
- npm

### Installation

1. Clone the repository:
```sh
git clone https://github.com/prpo-app/AppUI.git
cd AppUI
```

1. Install dependencies:
```sh
npm install
```
2. Start the development server:
```sh
npm start
```

The application will be available at `http://localhost:8080`

## Available Scripts

- `npm start` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production builds
